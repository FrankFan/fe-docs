Exampels

以著名的TodoMVC作为学习案例，分析和实践vue的用法。

todomvc提供了一个Application Specification规格说明书来描述App的结构和功能等。

同时，还提供了TodoMVC App Template模板，相当于规定了html文档结构。

最后，如果不想关心css样式的话，可以直接使用todomvc-app-css，用法极其简单，通过npm安装 npm install todomvc-app-css 后引入到页面中，就不用关心样式了，重心放在js如何实现就好了。
